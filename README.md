# Features

Transactions: date, account source, account destination, description, credit, debit

- List transactions
- Add transactions
- Update transactions

Data are stored in Database

# Usage

    python -m src.projet_budget

    [command] [args]

    transactions
        --list -l 
        --add -a 
        --update -u
