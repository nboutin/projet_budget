# Features

Transactions: date, account source, account destination, description, credit, debit

- List transactions
- Add transactions
- Remove transactions
- Update transactions

Data are stored in Database

# Usage

    python -m src.projet_budget

    [command] [args]

    transactions
        -l, --list
        -a, --add
        -r, --remove
        -u, --update
